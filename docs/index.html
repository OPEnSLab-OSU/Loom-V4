Welcome to Loom 4

Loom is a modular, open-source Arduino library designed to accelerate Internet of Things (IoT) prototyping for environmental, agricultural, and ecological sensing projects.

üéØ Project Purpose

Researchers and citizen scientists often struggle with the complexity and cost of deploying custom sensor networks in the field. Loom provides a plug-and-play framework that simplifies sensor integration, wireless communication, and data collection.

Who is it for?

  • Environmental scientists
  • Agricultural researchers
  • Educators and students

üîß Features and Benefits

  • Supports I2C, Serial, LoRa, MQTT, and SD logging
  • Flexible configuration via JSON metadata
  • Lightweight and low-power for remote deployment
  • Well-documented, open-source, and extendable

üìê Technical Architecture

Loom uses a modular, object-oriented design. Devices register sensors and communication modules through the Manager class, and data is routed through enabled services like MQTT or SD logging. Metadata is attached to every dataset for traceability.
